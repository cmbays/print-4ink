Reusable Components

Component Description

EmailPreviewModal Email preview dialog with To/Subject/Body fields. Mock send with toast confirmation.
StatusBadge Reusable badge with semantic color mapping per quote status (Draft=muted, Sent=action, Accepted=success, Declined=error, Revised=warning).
