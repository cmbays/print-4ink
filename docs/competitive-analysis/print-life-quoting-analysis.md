## UI Pattern Observations

### Garment Selection
- [ ] Visual catalog (swatches/images) or list?
- [ ] Searchable by SKU, style, brand, color?
- [ ] Shows size availability?
- [ ] Shows pricing during selection?
- [ ] Inline selection or modal?

### Quantity & Size Entry
- [ ] Single number field or size breakdown grid?
- [ ] If grid: S, M, L, XL, XXL, etc. as rows with qty columns?
- [ ] Inline edit or modal?
- [ ] Real-time validation or batch validation?

### Print Locations & Colors
- [ ] Multi-select dropdown, buttons, or free-form text?
- [ ] Locations are: Front, Back, Left Chest, Right Chest, Sleeves, Full Back, Custom?
- [ ] Color count field per location or global?
- [ ] Pre-set templates for common combinations?

### Pricing
- [ ] Auto-calculated or manual entry?
- [ ] Formula visible to user?
- [ ] Real-time updates or batch calculation?
- [ ] Settable per-line or global setup fees?
- [ ] Unit price, line total, subtotal, grand total all visible?

### Artwork Handling
- [ ] Required to create quote, or optional?
- [ ] Upload inline or modal?
- [ ] File size limits, format restrictions visible?
- [ ] Preview thumbnail in quote?

### Quote Submission
- [ ] "Send to Customer" button or manual email?
- [ ] Customer receives PDF, link, both?
- [ ] Customer needs login or public link?
- [ ] Auto-converts to job or manual conversion?
