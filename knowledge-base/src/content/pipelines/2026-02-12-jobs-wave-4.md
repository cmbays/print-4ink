## Wave 4A — Job Detail Command Center

`/jobs/[id]` -- The full job management interface.

### JobHeader
- Job number, title, customer name, service type badge
- Lane badge with lane-change dropdown
- Risk indicator, due date, priority badge
- Block/unblock quick action with reason tracking

### QuickActionsBar
- Move Lane (dropdown with all 5 lanes)
- Mark Blocked / Unblock (with block reason dialog)
- Print Job Sheet (placeholder for Phase 2)

### TaskChecklist
- Canonical tasks auto-populated by service type (screen-print: 8 tasks, DTF: 6, embroidery: 7)
- Toggle completion with optimistic UI
- Task progress bar with percentage
- Add custom tasks via inline input (`AddCustomTaskInput`)

### JobDetailsSection
- Garment details: style, brand, color, sizes breakdown
- Print locations: position, color count, mesh count, notes
- Order summary: quantity, total, dates

### NotesFeed
- Chronological notes with author, timestamp, type badges (general, issue, update)
- Add new note with type selector
- `NoteItem` component with relative timestamps

### LinkedEntitiesSection
- Source quote link with status and total
- Invoice links with status badges and amounts
- Customer link

### BlockReasonBanner
- Prominent banner when job is blocked
- Shows reason, blocked date, unblock button

## Wave 4B — Jobs List

`/jobs` -- Table view for managing all jobs.

### JobsDataTable
- TanStack Table with 9 columns: Job #, Title, Customer, Service Type, Lane, Risk, Due Date, Progress, Total
- Column sorting (click headers)
- Risk filter via `ColumnHeaderMenu`
- Search by job number, title, or customer name
- Row click navigates to job detail
- Quick actions: Move Lane, Block/Unblock
- Badge rendering: `ServiceTypeBadge`, `LaneBadge`, `RiskIndicator` inline

### Page Layout
- Breadcrumbs: Home > Jobs
- View toggle: List / Board (URL-driven)
- Filter + search bar
- `MoveLaneDialog` for bulk lane changes
